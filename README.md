# crime-economic-analysis
# Crime and Economic Factors Analysis

## Overview
This project analyzes the relationship between crime rates and economic and social variables using real-world GDP data.

## Data Sources
- GDP data: World Bank  
- Other variables (crime rate, unemployment, ELF): simulated for demonstration  

## Methodology
- Data cleaning and preparation  
- Correlation analysis  
- Data visualization  
- Linear regression (OLS)  

## Key Insights
- Economic and social variables influence crime rates  
- Ethnic diversity (ELF) shows a relationship with crime  
- Regression analysis provides additional explanatory insights  

## Academic Context
This project is inspired by my undergraduate thesis on socio-economic determinants of crime using econometric models.

## Tools Used
- Python  
- Pandas  
- Seaborn  
- Matplotlib  
- Statsmodels
